# 훅 최적화 가이드

## 훅이란?

쇼츠 영상의 첫 1~3초. 시청자가 스크롤을 멈추고 영상을 볼지 결정하는 순간.

## 4가지 훅 전략

### 1. 직접 선언형
사물이 자신을 직접 소개하며 반전을 만든다.
> "나는 연필인데, 할 말이 있어."
> "나는 오늘 버려졌어."

### 2. 감정 충격형
감정적 반응을 즉시 유발한다.
> "연필이 울고 있다면?"
> "당신이 버린 그 우산, 아직 거기 있어."

### 3. 공감 질문형
시청자의 경험에 직접 호소한다.
> "매일 쓰면서 한 번도 안 본 것."
> "지금 주머니 속에 뭐가 있어?"

### 4. 반전 사실형
알려진 사실을 뒤집어 호기심을 건다.
> "연필은 깎일수록 강해진다."
> "우산은 비를 막는 게 아니야."

## 훅 평가 기준 (5점 만점)

| 기준 | 설명 |
|------|------|
| 멈춤력 | 1초 안에 스크롤 멈추는가? |
| 정체 암시 | 사물이 말한다는 걸 느끼게 하는가? |
| 감정 유발 | 호기심/불편/공감 중 하나가 있는가? |
| 길이 적합 | 15자 이내인가? (3초 TTS) |
| 본문 연결 | 뒤의 대본과 자연스럽게 이어지는가? |

## 피해야 할 훅

- "안녕하세요, 저는 ___입니다" → 교과서적, 지루함
- 설명형 시작 → "오늘 소개할 것은..." 금지
- 너무 긴 문장 → 3초 초과 = 실패
- 클릭베이트 → 내용과 무관한 과장 금지
