# 사물 선정 가이드

## 좋은 사물의 조건

1. **일상성** — 누구나 매일 접하는 사물 (연필, 컵, 신발)
2. **감정 투사 가능** — 사람의 감정을 대입할 수 있는 것
3. **서사 잠재력** — "탄생 → 사용 → 버려짐" 서사가 자연스러운 것
4. **시각적 매력** — 클로즈업 촬영 시 텍스처가 흥미로운 것
5. **보편적 공감** — 문화/연령 무관하게 알아보는 것

## Thinking OS 적용 예시

### 연필
| 단계 | 내용 |
|------|------|
| 현상 | 매일 깎이고, 짧아지고, 결국 버려진다 |
| 패턴 | 사람들은 잘 써질 때만 연필을 의식한다 |
| 구조 | "도구는 기능이 전부"라는 효율 논리 |
| 전제 해체 | 연필이 짧아지는 건 실패가 아니라 헌신의 증거다 |
| 재설계 | 짧은 연필을 버리지 않고 끝까지 쓰는 관계 |

### 우산
| 단계 | 내용 |
|------|------|
| 현상 | 비 올 때만 찾고, 갠 날엔 구석에 던져진다 |
| 패턴 | 필요할 때만 꺼내고, 필요 없으면 잊는다 |
| 구조 | "도움은 위기일 때만 가치 있다"는 전제 |
| 전제 해체 | 맑은 날의 우산도 존재의 의미가 있지 않을까 |
| 재설계 | 쓰지 않는 날에도 곁에 두는 관계 |

## 피해야 할 사물

- 추상적 개념 (사랑, 시간) → 사물이 아님
- 너무 큰 것 (건물, 자동차) → 클로즈업 어려움
- 논란적 것 (무기, 담배) → 쇼츠 정책 위반 리스크
- 너무 특수한 것 (현미경, 만년필) → 공감대 좁음
